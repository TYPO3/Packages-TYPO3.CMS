Extbase
-------------------------------------------------------------------------------

* Support of custom language fallbacks

Please be aware of the fact that the content_fallback setting respects the
already documented customizable language fallback feature like
"sys_language_mode = content_fallback; 1,2" much more completely now.
In general this means that you can define e.g. German as fallback language
for a different region like e.g. Austria instead of the site-wide English
default language. The setting is taken into account independently on page and
content level. The latter only if related extensions are using the new
API method "getRecordOverlayWithFallback" instead of the
old (deprecated) "getRecordOverlay".

-------------------------------------------------------------------------------
Fluid
-------------------------------------------------------------------------------

* Image view helper does not render title tag by default

In previous versions of fluid the image view helper always rendered the
title attribute. If not set, the value of the required alt attribute was set as
title.
This fallback was removed with version 6.2. If not specifically set, title
is not rendered anymore.

Example:
  Fluid Tag
    <f:image src="{file}" alt="Alt-Attribute" />
  will render
    <img src="fileadmin/xxxx.jpg" alt="Alt-Attribute" />
  and not
    <img src="fileadmin/xxxx.jpg" alt="Alt-Attribute" title="Alt-Attribute" />
